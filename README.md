# 🖼️ Image Projector

> 專為課堂、講座與簡報設計的圖片投影工具  
> 雙螢幕即時同步 · 畫筆標註 · 遮罩覆蓋 · 聚光燈聚焦

---

## ✨ 功能特色

### 🗂️ 圖片瀏覽
- 左側 **資料夾樹狀檢視**，支援展開／收合，顯示各資料夾的圖片數量
- 底部 **縮圖列**，序號補零、非同步載入，支援滑鼠點擊與鍵盤快速切換
- 支援格式：`.jpg` `.jpeg` `.png` `.gif` `.bmp` `.webp` `.tif` `.tiff` `.ico` `.svg` `.wmf` `.emf`
- 拖曳資料夾或圖片直接開啟
- **書籤管理**：加入常用資料夾，可設定固定啟動資料夾

### 🔍 縮放與視角
| 功能 | 說明 |
|------|------|
| 內切／外切 | 完整顯示或填滿視窗 |
| 固定百分比 | 25% / 50% / 75% / 100% / 150% / 200% |
| 滑鼠滾輪縮放 | 以滑鼠游標為中心縮放 |
| Ctrl + 拖曳 | 框選後放大至選取範圍 |
| 雙指觸控縮放 | 以兩指中心為縮放基準 |
| 左鍵拖曳 | 平移圖片 |
| 雙擊 | 鎖定內切（符合視窗大小） |
| 方向鍵 | 平移圖片（每步 10 螢幕像素） |
| 旋轉 | 左轉／右轉 90°（僅顯示，不修改檔案） |
| 視窗比例快速設定 | 4:3 / 3:2 / 16:10 / 16:9 |

### 📽️ 雙螢幕投影
- 一鍵將圖片投影至第二螢幕（無框全螢幕黑底）
- **即時同步**：縮放、平移、旋轉、筆跡、遮罩全部同步顯示
- 下拉選單選擇目標螢幕，支援多螢幕環境

### ✏️ 畫筆標註（不修改原始檔案）
| 工具 | 說明 |
|------|------|
| ✏️ 手繪 | Cardinal Spline 平滑曲線 |
| ╱ 直線 | |
| □ / ■ 矩形 | 空心／實心 |
| ○ / ● 圓形 | 空心／實心 |
| → 箭頭 | |
| T 文字 | 自訂字型與大小 |
| ⌫ 橡皮擦 | 點擊刪除 / Ctrl+拖曳框選刪除 |

- 自訂**顏色**（12 個快捷色塊 + 色盤任意選）與**透明度**（0～100%）
- 自訂**筆跡粗細**（NUD 或 6 個快捷按鈕）
- **筆跡自動儲存**：切換圖片時以 XML 旁置檔（`*.strokes.xml`）持久化，下次開啟自動還原

### 🔤 T 文字自動遞增
在文字框輸入以下格式，每次點擊後自動遞增：

| 格式 | 說明 |
|------|------|
| `{1}` | 阿拉伯數字 |
| `{一}` | 中文數字 |
| `{壹}` | 中文大寫數字 |
| `{A}` / `{a}` | 大寫／小寫英文字母（26 進位） |
| `{甲}` | 天干（10 循環） |
| `{子}` | 地支（12 循環） |
| `{甲子}` | 六十甲子（60 循環） |
| `{A,B,C,D}` | 右鍵顯示快速功能表，點擊位置繪製選取項 |

### ▦ 遮罩覆蓋
- 在圖片上新增半透明**黑色矩形遮罩**，用於遮蓋局部內容
- 支援移動、縮放（8 個控制點）、多選（Ctrl 點擊）
- **滾輪**調整選取遮罩透明度
- **Delete** 刪除選取遮罩
- 遮罩固定於圖片座標，跟隨縮放／平移／旋轉，投影同步顯示
- 遮罩連同筆跡一併儲存至 XML 旁置檔

### 🔦 聚光燈
- 滑鼠周圍圓形高亮，四周半透明暗化，引導觀眾視線集中

### 🖼️ 空白畫布
- 新建指定尺寸與背景色的空白 PNG 畫布（最大支援 8192×4320）
- 畫布儲存於程式目錄的 `Canvas\` 子資料夾，可透過下拉選單快速切換

### 💾 合併輸出
- 將底圖、筆跡（粗細依縮放比例補償）、遮罩合併輸出為 PNG 檔案

## ⌨️ 快捷鍵總覽

### 縮圖區
| 按鍵 | 動作 |
|------|------|
| `← / → / ↑ / ↓` | 選取上／下一張圖片 |
| `Home / End` | 跳到第一張／最後一張 |
| `PageUp / PageDown` | 往前／往後翻一頁並選圖 |
| 滾輪 | 橫向捲動縮圖列 |

### 圖片區
| 按鍵 | 動作 |
|------|------|
| `Space` | 瀏覽 ↔ 畫筆（或遮罩編輯）模式切換 |
| `Alt + ←` / `Alt + →` | 上一個／下一個資料夾 |
| 滾輪 | 縮放（以游標為中心） |
| 左鍵拖曳 | 平移圖片 |
| 雙擊 | 鎖定內切 |
| `Ctrl` + 左鍵拖曳 | 框選放大 |
| 方向鍵 | 平移（每步 10px） |
| 右鍵 | 快速功能表 |
| `Escape`（投影視窗） | 隱藏投影 |

## 🚀 安裝與執行

本程式為**免安裝**的單一執行檔，無需安裝程序。

1. 下載 Release 頁面的壓縮檔並解壓縮
2. 執行 `Image_Projector.exe`
3. （選用）在圖片或資料夾上按右鍵 → **開啟方式** → 選擇 `Image_Projector.exe`，即可直接從檔案總管開啟

> 設定（視窗位置、書籤、上次資料夾等）自動儲存於 Windows Registry  
> 筆跡與遮罩儲存於圖片旁的 `*.strokes.xml` 旁置檔
